# flightplan

## Project Description
Flightplan is a web application designed to help users manage their tasks and events efficiently. It features a user-friendly interface that allows users to log in, view upcoming events, and track their progress. The application dynamically displays a navigation bar and footer based on the user's login status.

## Project Setup
To set up the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/flightplan.git
   cd flightplan
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Application
To run the application in development mode, use the following command:
```bash
npm run serve
```

### Compiling for Production
To compile and minify the application for production, run:
```bash
npm run build
```

### Linting and Fixing Files
To lint and fix files, use:
